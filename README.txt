Nombre del proyecto: Casa Domótica

Descripción: A través de la domótica, un sistema automatizado que se monitorea, 
se permite que el usuario supervise el reciclaje del agua de manera remota mediante una aplicación móvil.

Nombres de los integrantes:

Dependencias necesarias:
React Navigation (para la navegación)
npm install @react-navigation/native
Stack Navigator (para la navegación entre pantallas)
npm install @react-navigation/stack
